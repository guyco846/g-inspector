# G-Inspector 📋

מערכת דיגיטלית להפקת דוחות פיקוח הנדסיים בפורמט E.SPIVAK.

## 🎯 מה האפליקציה עושה?

אפליקציית web שמאפשרת למפקחים בשטח:
- ✅ ליצור דוחות סיור עם תמונות, מיקומים ותיאורים
- ✅ להשתמש בבנק תבניות מוכן עם משפטים נפוצים
- ✅ להזין טקסט באמצעות הקלטה קולית
- ✅ להפיק PDF זהה לפורמט המקורי של ספיבק
- ✅ לשמור גיבוי אוטומטי בגוגל דרייב
- ✅ לעבוד ללא אינטרנט (Offline)

## 🚀 שימוש

פשוט כנס לכתובת: `https://YOUR-USERNAME.github.io/g-inspector`

(החלף את `YOUR-USERNAME` בשם המשתמש שלך ב-GitHub)

## ⚙️ הגדרת גוגל דרייב

האפליקציה עובדת בלי הגדרה - אבל לחיבור לגוגל דרייב, ערוך את הקובץ `config.js`:

```javascript
window.GI_CONFIG = {
    GOOGLE_CLIENT_ID: "השם-שקיבלת.apps.googleusercontent.com"
};
```

הוראות מפורטות בקובץ `מדריך_התקנה.pdf`.

## 📁 מבנה הפרויקט

```
g-inspector/
├── index.html          ← האפליקציה (קובץ יחיד 770KB)
├── config.js           ← הגדרות (Client ID של גוגל)
├── README.md           ← הקובץ הזה
└── מדריך_התקנה.pdf     ← הוראות מלאות עם צילומי מסך
```

## 🛠 טכנולוגיות

- HTML5, CSS3, Vanilla JavaScript (ללא תלויות)
- jsPDF + html2canvas להפקת PDF
- Google Drive API v3 לסנכרון
- Web Speech API להקלטה קולית
- LocalStorage לשמירה אופליין

## 🔐 פרטיות

- כל הנתונים נשמרים על המכשיר שלך (LocalStorage)
- הקבצים שמועלים לגוגל דרייב נשמרים בחשבון שלך בלבד
- אין שרת חיצוני - אנחנו לא רואים שום דבר

## 📞 תמיכה

אם משהו לא עובד:
1. בדוק את `מדריך_התקנה.pdf`
2. ודא שה-Client ID נכון ב-`config.js`
3. ודא שה-domain של GitHub Pages מורשה ב-Google Cloud
